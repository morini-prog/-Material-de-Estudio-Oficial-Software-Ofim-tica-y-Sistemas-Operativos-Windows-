// Bloque 01

Introducción al Software
e Ingeniería de Sistemas

Todo sobre la parte intangible de la computación: qué es, cómo se construye y cómo se clasifica.

🖥️
Componente físico
Hardware

Todo aquello que es tangible en un dispositivo. Lo que se puede tocar: teclado, pantalla, cables, chips, mouse, placa de video.

💾
Componente lógico
Software

La parte intangible (digital) del sistema. Conjunto de instrucciones, datos y programas lógicos que le dicen al hardware qué hacer. Actúa como traductor entre humanos y máquinas.

Ciclo de Vida del Software

¿Cómo se fabrica un programa?

Los programas no se crean al azar. Se sigue un proceso estructurado con etapas ordenadas. Hacé clic para expandir cada fase:

01
Análisis de Requisitos
Se identifican las necesidades del usuario y se define qué debe hacer el programa. Se divide en:

Requisitos Funcionales: Las funciones específicas del programa (ej. "el botón debe imprimir un archivo").
Requisitos No Funcionales: Características generales como velocidad, seguridad y compatibilidad.
02
Diseño
Se planea la estructura del código y cómo se verá la interfaz (pantalla). Incluye diseño de base de datos, arquitectura del sistema y flujos de usuario.
03
Codificación
Los programadores escriben las instrucciones usando lenguajes de programación (Java, Python, C++, JavaScript, entre otros).
04
Pruebas (Testeo)
Se ejecuta el programa para detectar y corregir errores (bugs) antes de que llegue a los usuarios finales. Incluye pruebas unitarias, de integración y de usuario.
05
Instalación / Paso a Producción
Se distribuye y se pone a funcionar en los equipos de los usuarios. En proyectos modernos se usan pipelines de CI/CD para automatizar este proceso.
06
Mantenimiento
Se actualiza el software para corregir fallos futuros o añadir nuevas funciones. Esta etapa puede durar años y consume la mayor parte del presupuesto de un proyecto de software a largo plazo.
Tip: Modelos de Desarrollo de Software

Los más preguntados en clase son el Modelo en Cascada (un paso después de otro de forma estricta), el Modelo en Espiral (evaluando riesgos de forma cíclica) y el Modelo Iterativo Incremental (construyendo el programa por partes).

Taxonomía

Los cuatro tipos de Software

Clasificados según su propósito técnico:

Categoría Propósito Principal Ejemplos
Software de Sistema Permite el control básico y manejo operativo del hardware. Es la "base" para que los demás programas funcionen. Windows, macOS, Android, iOS, Linux
Software de Aplicación Diseñado para realizar tareas específicas de estudio, trabajo, entretenimiento o comunicación. Word, Chrome, WhatsApp, Netflix, videojuegos
Software de Programación Herramientas que usan los desarrolladores para escribir y crear nuevos programas. Visual Studio, Eclipse, Git, compiladores
Software de Utilidad Herramientas de mantenimiento que protegen, analizan u optimizan el rendimiento del equipo. Antivirus, limpiadores de archivos, desfragmentadores
Alerta de Examen: Malware vs Software Defectuoso

El malware es diseñado a propósito con fines delictivos para dañar o espiar. El software defectuoso simplemente contiene fallos involuntarios de programación. No los confundas.

// Bloque 02

Ofimática y Suites
de Oficina

Las herramientas digitales para automatizar el trabajo cotidiano: escribir, calcular, presentar, comunicar.

💼

¿Qué es la Ofimática?

Combinación de Oficina + Informática. Conjunto de programas para automatizar y optimizar las tareas habituales del ámbito laboral o académico: escribir textos, hacer cuentas, armar diapositivas, gestionar correos.

Microsoft 365
Word · Excel · PowerPoint · Outlook · OneNote
Propietario · Pago

Modelo: Software propietario bajo suscripción mensual/anual. Incluye aplicaciones de escritorio locales y versiones web. Estándar de facto en ambientes corporativos y gubernamentales.

Fortalezas

Las apps locales más potentes del mercado. Ideal para hojas de cálculo complejas y macros en Excel. Excelente integración empresarial con Microsoft Intune y Entra ID.

Debilidades

Requiere pago constante. Si caduca la suscripción, las apps entran en modo solo lectura.

Google Workspace
Docs · Sheets · Slides · Gmail · Meet
100% Nube · Suscripción

Modelo: Software propietario, enfocado 100% en la nube. Se ejecuta desde el navegador sin instalación local. Orientado a la colaboración en tiempo real.

Fortalezas

Líder absoluto en edición simultánea entre múltiples usuarios. Interfaz muy sencilla de aprender. Historial de versiones automático.

Debilidades

Capacidades limitadas sin conexión a internet. Menor potencia que Excel en hojas de cálculo muy avanzadas.

LibreOffice
Writer · Calc · Impress · Draw · Base
Gratuito · Código Abierto

Modelo: Software de código abierto, gratuito, bajo Licencia Pública de Mozilla (MPLv2). Sucesor del antiguo OpenOffice. Desarrollado por The Document Foundation.

Fortalezas

Completamente gratuito, sin límite de instalaciones. No requiere internet. Altamente portátil (puede ejecutarse desde USB). Respeta la privacidad.

Debilidades

Interfaz visualmente menos moderna. Sin colaboración en tiempo real nativa. Puede haber diferencias al abrir archivos .docx creados en Word.

Sistemas Operativos

El software más importante de cualquier computadora: el que hace que todo lo demás funcione.

Concepto Clave

El Sistema Operativo (SO)

El software principal e indispensable que sirve como puente de comunicación directo entre el usuario y las piezas físicas de la computadora. Sin él, la computadora no sabría cómo encender ni cómo responder al teclado.

Fuentes de profundización: Concepto.de · GoDaddy

Las 6 Funciones Principales del SO

Los profesores adoran evaluar estas funciones. Hacé clic para expandir:

Administra el procesador (CPU) asignando tiempos y prioridades a las tareas activas. Permite que múltiples procesos compartan el CPU mediante técnicas de scheduling.
Controla la memoria RAM, decidiendo cuánto espacio le presta a cada aplicación abierta. También gestiona la memoria virtual en el disco duro cuando la RAM se agota.
Permite crear, renombrar, guardar, organizar y borrar carpetas y archivos en el almacenamiento. Gestiona los sistemas de archivos (NTFS, FAT32, ext4).
Controla la entrada y salida de datos de accesorios externos (teclados, mouse, impresoras, cámaras) mediante controladores o drivers.
Regula los usuarios y los accesos autorizados al equipo. Implementa políticas de contraseñas, encriptación y control de acceso a archivos y carpetas.
Proporciona un entorno amigable para que el usuario controle todo de forma sencilla. Puede ser gráfico (GUI, como Windows) o de comandos (CLI, como una terminal de Linux).

Clasificación de los S.O.

Se agrupan según tres dimensiones básicas:

Por cantidad de usuarios
Monousuario
Solo permite ejecutar programas de un usuario a la vez.
Multiusuario
Permite que múltiples usuarios ejecuten simultáneamente sus cuentas y programas.
Por cantidad de tareas
Monotarea
Solo procesa una tarea a la vez; para otra, debes cerrar la actual.
Multitarea
Ejecuta múltiples programas al mismo tiempo (navegar mientras escuchás música).
Por asignación de recursos
Centralizado
Trabaja usando los recursos de una sola computadora física.
Distribuido
Coordina los recursos de varias computadoras conectadas en red al mismo tiempo.

Interfaz y Ventanas
de Windows

Windows utiliza una Interfaz Gráfica de Usuario (GUI): interactuás mediante iconos, botones y menús visuales en lugar de comandos de texto.

El Escritorio e Iconos

El Escritorio es el área de pantalla que aparece al arrancar Windows. Técnicamente es una carpeta del sistema de archivos que sirve para mantener a mano tus herramientas de trabajo diarias.

📄
Archivo Real
⚠ Sin flecha — peligro
Es el archivo original guardado en el Escritorio. Si lo borrás, se elimina permanentemente del equipo.
📄↗
Acceso Directo
✓ Con flecha — seguro
Es solo un puntero al archivo real. Si lo borrás, el archivo original permanece intacto en el disco.
Alerta de Examen: Icono Real vs Acceso Directo

La diferencia está en la pequeña flecha en la esquina inferior izquierda. Con flecha = seguro de borrar. Sin flecha = es el archivo real.

Partes de una Ventana

Cada carpeta o aplicación se despliega en un recuadro interactivo llamado Ventana. Hacé clic para conocer sus partes:

En la parte superior; muestra el nombre del archivo y programa. Doble clic: maximiza/restaura la ventana. Clic sostenido: arrastrá para moverla. Referencia oficial: Microsoft Learn.
Esquina superior izquierda. Un clic abre el menú de control; doble clic cierra la ventana de inmediato.
Esquina superior derecha:

Minimizar (−): Oculta la ventana en la barra de tareas sin cerrarla.
Maximizar / Restaurar (▢): Ocupa toda la pantalla o vuelve al tamaño anterior.
Cerrar (✕): Cierra la aplicación definitivamente.
En el borde derecho o inferior cuando el contenido supera el tamaño de la ventana. Se arrastran con el mouse para navegar vertical u horizontalmente.
Parte inferior de la ventana. Muestra información contextual pasiva: número de páginas, estado de carga, ruta actual. No es interactiva.
Barra de Tareas, Menú Inicio y Bandeja

Atraviesa toda la parte inferior de la pantalla. Está compuesta por varios bloques funcionales:

🚀
Menú Inicio
Al presionarlo (extremo izquierdo) despliega accesos directos para iniciar todos los programas instalados.
⚡
Barra de Inicio Rápido
Iconos de aplicaciones abiertas o fijadas. Permanece visible incluso si otras ventanas cubren el escritorio.
🕐
Bandeja de Sistema (System Tray)
A la derecha (junto al reloj). Muestra programas que trabajan en segundo plano: antivirus, volumen, Wi-Fi, batería.
👁️
Mostrar Escritorio
Pequeño recuadro en la esquina inferior derecha. Un clic minimiza todas las ventanas al instante.

Atajos de Teclado
Esenciales

Dominá las combinaciones que te ahorran horas de trabajo. Filtrá por nombre o categoría.

Edición
Copiar Ctrl+C

Copia la selección al portapapeles de forma temporal.

Edición
Cortar Ctrl+X

Elimina la selección de su lugar original para moverla a otro sitio.

Edición
Pegar Ctrl+V

Pega lo copiado o cortado en la posición del cursor.

Edición
Deshacer Ctrl+Z

Deshace la última acción. ¡Salvavidas de errores!

Edición
Rehacer Ctrl+Y

Rehace la última acción cancelada con Deshacer.

Edición
Seleccionar Todo Ctrl+A

Selecciona todo el texto del documento o todos los archivos de una carpeta.

Edición
Guardar Ctrl+S

Guarda el archivo actual. Usalo constantemente para no perder trabajo.

Edición
Buscar Ctrl+F

Abre el cuadro de búsqueda en la aplicación o navegador activo.

Navegación
Cambiar Ventana Alt+Tab

Cambia de ventana rápidamente sin usar el mouse.

Navegación
Cerrar App Alt+F4

Cierra la aplicación activa. En el escritorio abre el menú de apagado.

Navegación
Mostrar Escritorio Win+D

Minimiza todo para ver el escritorio. Al pulsar de nuevo restaura todo.

Navegación
Explorador de Archivos Win+E

Abre directamente el Explorador de Archivos de Windows.

Navegación
Bloquear PC Win+L

Bloquea la sesión para proteger tu computadora de miradas curiosas.

Navegación
Administrador de Tareas Ctrl+Shift+Esc

Abre el Administrador de Tareas directamente para ver/matar procesos.

Navegación
Buscar en Windows Win+S

Abre la búsqueda de Windows para encontrar apps, archivos o configuraciones.

Especiales
Cambiar Nombre F2

Renombra el archivo o carpeta seleccionado de inmediato.

Especiales
Actualizar / Recargar F5

Recarga la pantalla activa o la página web abierta.

Especiales
Eliminar Permanente Shift+Supr

Borra el archivo sin enviarlo a la Papelera de Reciclaje. ¡Irreversible!

Especiales
Captura de Pantalla Win+Shift+S

Abre la herramienta Recortes para capturar una zona de la pantalla.
