Bloque 1: Introducción al Software e Ingeniería de Sistemas

Concepto Clave: Hardware vs. Software

Hardware (La infraestructura física)

Es todo aquello que es tangible en una computadora o dispositivo electrónico; es decir, "lo que se puede tocar" (teclado, pantalla, cables, chips, mouse).

Software (La dimensión lógica)

Es la parte intangible (digital) del sistema. Consiste en el conjunto de instrucciones, datos y programas lógicos que le dicen al hardware exactamente qué hacer y cómo hacerlo. Actúa como un traductor para que los humanos interactuemos con la máquina sin tener que saber código binario.

El Ciclo de Vida del Software (¿Cómo se fabrica un programa?)

En la ingeniería de software, los programas no se crean al azar; se sigue un proceso estructurado dividido en etapas ordenadas. Haz clic en cada una para desplegar los detalles:

Se identifican las necesidades del usuario final y se define qué debe hacer el programa. Se divide en:
  • Requisitos Funcionales: Las funciones específicas del programa (ej. "el botón debe permitir imprimir un archivo").
  • Requisitos No Funcionales: Características generales del sistema (ej. velocidad del programa, seguridad, compatibilidad).
Se planea la estructura del código y cómo se verá la interfaz (pantalla).
Los programadores escriben las instrucciones usando lenguajes de programación (Java, Python, C++).
Se ejecuta el programa para detectar y corregir errores (bugs) antes de que llegue a los usuarios.
Se distribuye y se pone a funcionar en los equipos de los usuarios.
Se actualiza el software para corregir fallos futuros o añadir nuevas funciones.

Tip de Estudio: Modelos de Desarrollo de Software

Para estructurar estas etapas se usan "modelos de desarrollo" de software. Los tres más preguntados en clase son el Modelo en Cascada (un paso después de otro de forma estricta), el Modelo en Espiral (evaluando riesgos de forma repetitiva) y el Modelo Iterativo Incremental (creando el programa por partes poco a poco).

Taxonomía: Los cuatro tipos de Software

Los programas se clasifican según su propósito técnico. Para conocer más sobre cómo se clasifican estos programas, puedes consultar la guía de tipos de software de Ceupe o revisar ejemplos prácticos en la web de GoDaddy.

Categoría Propósito Principal Ejemplos
Software de Sistema Permite el control básico y manejo operativo del hardware. Es la "base" para que los demás programas puedan funcionar. Microsoft Windows, macOS, Android, iOS.
Software de Aplicación Diseñado para realizar tareas específicas de estudio, trabajo, entretenimiento o comunicación. Word, Google Chrome, WhatsApp, Netflix, videojuegos.
Software de Programación Herramientas que usan los desarrolladores para escribir y crear nuevos programas. Visual Studio, Eclipse, Git, compiladores.
Software de Utilidad Herramientas de mantenimiento que protegen, analizan u optimizan el rendimiento físico del equipo. Antivirus, limpiadores de archivos temporales, desfragmentadores.

Alerta de Examen: Malware vs Software Defectuoso

No confundas Software Malicioso (Malware) con Software Defectuoso. El malware es diseñado a propósito con fines delictivos para dañar o espiar. El software defectuoso simplemente contiene fallos involuntarios de programación creados por error humano.

Bloque 2: Ofimática y Suites Ofimáticas Existentes

CONCEPTO CLAVE: ¿Qué es la Ofimática?

Es la combinación de palabras de Oficina e Informática. Se refiere al conjunto de programas informáticos que se utilizan para automatizar, optimizar y realizar tareas habituales del ámbito laboral o académico (escribir textos, hacer cuentas, armar diapositivas, gestionar correos).

ESQUEMA GENERAL: SUITES OFIMÁTICAS

Microsoft 365

Propietario • Offline/Online • Muy potente

Google Workspace

Web-First • Súper colaborativo

LibreOffice

Gratis/Libre • Código Abierto

Comparativa: Las 3 Grandes Suites de Oficina

1. Microsoft 365 (Word, Excel, PowerPoint)

Propietario / Pago

Modelo: Software propietario bajo suscripción de pago (alquiler mensual/anual). Puedes revisar las diferencias detalladas de administración y rendimiento en este análisis sobre Microsoft 365 vs Google Workspace.

Fortalezas

Ofrece las aplicaciones locales de escritorio más potentes del mercado (ideales para hojas de cálculo complejas y macros pesadas en Excel). Excelente integración con herramientas de seguridad empresarial y administración de redes (como Microsoft Intune y Entra ID).

Debilidades

Requiere un pago constante; si dejas de pagar, las apps entran en modo lectura.

2. Google Workspace (Google Docs, Sheets, Slides)

100% Nube / Suscripción

Modelo: Software propietario bajo suscripción de pago, enfocado 100% en la nube (ejecutado desde el navegador). Puedes consultar la comparativa de precios y funciones actualizadas en la web de Badgerlayer.

Fortalezas

Es la herramienta reina para la edición simultánea en tiempo real entre varios estudiantes o compañeros de trabajo. Interfaz limpia, sencilla y muy fácil de aprender a usar sin conocimientos de TI.

Debilidades

Capacidades limitadas cuando no se cuenta con conexión a internet y menor potencia en hojas de cálculo muy avanzadas en comparación con Excel.

3. LibreOffice (Writer, Calc, Impress)

Gratuito / Código Abierto

Modelo: Software de código abierto, gratuito, bajo la Licencia Pública de Mozilla (MPLv2). Es el sucesor del antiguo OpenOffice. Puedes examinar la tabla técnica completa en la Document Foundation Wiki o explorar debates sobre estas herramientas en los hilos de Reddit.

Fortalezas

Es gratuito; puedes instalarlo en tantas computadoras como desees. No dependes de internet, es altamente portátil (se puede ejecutar desde un USB sin instalación) y respeta completamente la privacidad de tus datos.

Debilidades

La interfaz visual suele verse un poco anticuada. No cuenta con herramientas nativas de colaboración en línea en tiempo real. Puede haber ligeras alteraciones visuales al abrir documentos creados originalmente en Microsoft Word (.docx).

Bloque 3: Sistemas Operativos y Fundamentos de Windows

CONCEPTO CLAVE: El Sistema Operativo (SO)

Es el software principal e indispensable que sirve como puente de comunicación directo entre tú (el usuario) y las piezas físicas de la computadora (el hardware). Sin un sistema operativo, la computadora no sabría cómo encender ni cómo responder al teclado.

Puedes estudiar en profundidad estas funciones básicas y componentes en el artículo sobre Concepto.de o aprender cómo elegir el más adecuado en el sitio de GoDaddy.

Las 6 Funciones Principales de un Sistema Operativo

Los profesores de informática adoran evaluar estas funciones. Haz clic en cada una para ver la explicación:

Administra el procesador (CPU) asignando tiempos y prioridades a las tareas activas.
Controla la memoria de acceso aleatorio (RAM), decidiendo cuánto espacio le presta a cada aplicación abierta.
Permite crear, renombrar, guardar, organizar y borrar tus carpetas y archivos en el almacenamiento.
Controla la entrada y salida de datos de accesorios externos (teclados, mouse, impresoras) mediante controladores o drivers.
Regula los usuarios y los accesos autorizados al equipo de manera protegida y restrictiva.
Proporciona un entorno amigable (visual o de comandos) para que el usuario controle todo de forma sencilla.

Clasificación de los Sistemas Operativos

Los sistemas operativos se dividen y agrupan según tres dimensiones básicas. Despliega cada una para estudiarla:

  • Monousuario: Solo permite ejecutar programas de un usuario a la vez.
  • Multiusuario: Permite que múltiples usuarios ejecuten simultáneamente sus cuentas y programas.
  • Monotarea: Solo puede procesar una sola tarea a la vez (si quieres hacer otra cosa, debes cerrar la actual).
  • Multitarea: Permite ejecutar múltiples programas al mismo tiempo (ej. navegar por internet mientras escuchas música).
  • Centralizado: Trabaja utilizando los recursos de una sola computadora física.
  • Distribuido: Permite usar y coordinar los recursos de más de una computadora conectada en red al mismo tiempo.

Bloque 4: Manejo de la Interfaz y Ventanas de Windows

Windows utiliza una Interfaz Gráfica de Usuario (GUI). Esto significa que, en lugar de escribir aburridas líneas de comandos de texto, interactúas mediante elementos visuales intuitivos como iconos, botones y menús.

El Escritorio de Windows e Iconos

El Escritorio: Es el área de pantalla grande que aparece apenas arranca el sistema. Técnicamente, es una carpeta más que se almacena en el sistema de archivos de Windows, y sirve para mantener a la mano tus herramientas de trabajo diarias.

Alerta de Examen: Distinguir Iconos Reales vs. Accesos Directos

Para evitar borrar tus tareas por error, debes saber diferenciarlos:

  • Iconos de Documentos (Archivos Reales): Son los archivos originales guardados en la carpeta del escritorio. Tienen el logotipo de su programa pero NO tienen ninguna flecha.
    Consecuencia: Si los borras, eliminas el archivo real de tu computadora definitivamente.
  • Accesos Directos (Atajos hacia aplicaciones): Son solo enlaces rápidos o "punteros" que apuntan al archivo ejecutable real que hace correr el programa. Se identifican porque tienen una pequeña flecha en la esquina inferior izquierda del icono.
    Consecuencia: Borrarlos es 100% seguro; el archivo o programa original sigue a salvo en tu disco duro.

Para leer una explicación detallada sobre cómo interactuar con estos elementos, visita la guía de la Interfaz de Windows.

Partes de las Ventanas de Windows

Cada carpeta o aplicación que ejecutas en Windows se despliega dentro de un recuadro interactivo llamado Ventana. Haz clic en cada parte para conocer sus detalles:

Se encuentra en la parte superior. Muestra el nombre del archivo y del programa abierto.

Truco de mouse: Al hacer doble clic sobre ella, la ventana se maximiza (ocupa toda la pantalla) o recupera su tamaño restaurado. También sirve para arrastrar y mover la ventana por el escritorio haciendo un clic sostenido.

Para ver las pautas oficiales de diseño de este componente, consulta el documento de Diseño de Barra de Título de Microsoft.

Ubicado en la esquina superior izquierda. Un solo clic abre el menú de control de ventana; un doble clic cierra la ventana de inmediato.
Son tres botones estándar para operar de forma manual el recuadro interactivo:
  • Minimizar (icono -): Esconde la ventana y la guarda en la barra de tareas en la parte inferior de la pantalla sin detener su ejecución.
  • Maximizar / Restaurar (icono ▢): Agranda la ventana para ocupar todo el monitor, o la devuelve a su tamaño normal.
  • Cerrar (icono X): Cierra la aplicación o el documento de forma definitiva.
Se ubican en el borde derecho o inferior cuando la información del documento excede el tamaño físico de la ventana. Se arrastran con el mouse para navegar por el contenido vertical u horizontalmente.
Ubicada en la parte inferior de la ventana. Muestra información contextual pasiva (como número de páginas, estado de cargas o rutas). No es interactiva mediante doble clic o clic derecho.

La Barra de Tareas, el Menú Inicio y la Bandeja

Atraviesa toda la parte inferior de la pantalla y consta de varios bloques funcionales:

El Menú Inicio

Al presionar su botón (extremo izquierdo), se despliega una lista de accesos directos para iniciar todos tus programas instalados.

La Barra de Inicio Rápido / Iconos

Ubicada al lado del menú inicio. Muestra iconos de aplicaciones abiertas o fijadas. Es sumamente útil porque permanece visible incluso si tienes otras ventanas encima del escritorio.

La Bandeja de Sistema (System Tray)

Ubicada a la derecha de la barra de tareas (junto al reloj). Muestra iconos de programas que trabajan de forma silenciosa en segundo plano (antivirus, volumen, conexión Wi-Fi, nivel de batería).

Botón Mostrar Escritorio

Un pequeño recuadro al final de la esquina inferior derecha. Al hacer un clic sobre él, minimiza inmediatamente todas las ventanas abiertas para que puedas ver el escritorio al instante.

Puedes aprender sobre cómo personalizar esta área en la guía de Soporte de Microsoft o repasar su concepto general en el Glosario de Lenovo.
